# 統計數字串列

Python 內建許多函式可供我們順利達成許多任務，如果我們想統計整年度業績最大值、最小值以及總和的話，該怎麼做呢？

#### 實驗室 1 - 統計該年度最大銷售額

```python
sales = [100, 400, 600, 300, 2000, 50, 300, 400, 500, 600, 700, 200]
print(max(sales))
```

使用 `max` 就可以了解到該串列內最大值。

#### 實驗室 2 - 統計該年度最小銷售額

```python
sales = [100, 400, 600, 300, 2000, 50, 300, 400, 500, 600, 700, 200]
print(min(sales))
```

使用 `min` 就可以了解到該串列內最小值。

#### 實驗室 3 - 統計該年度總銷售額

```python
sales = [100, 400, 600, 300, 2000, 50, 300, 400, 500, 600, 700, 200]
print(sum(sales))
```

使用 `sum` 就可以了解到該串列加總為何。
